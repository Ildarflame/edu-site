## Contributing

Contributions are welcome! If you know of a deal that should be on this list:

1. Fork this repository
2. Add your deal to the appropriate category following the format: `- [Name](url) -- Description. **Value:** amount`
3. Verify the deal is currently active and available
4. Submit a pull request

Please ensure:
- The deal offers genuine value (free or significantly discounted)
- The URL is a direct link to the program page
- The value estimate is accurate and up to date
- The deal is available to at least one of: students, startups, or OSS maintainers
