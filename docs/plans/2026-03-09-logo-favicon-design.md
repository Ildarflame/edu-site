# Logo & Favicon Design

**Symbol:** Stylized lightning bolt — simple geometric shape, readable at 16x16.

**Colors:** Gradient from `#f97316` (orange-500) to `#f59e0b` (amber-500). Rounded ends, slight tilt for dynamism.

**Deliverables:**

| Artifact | Format | Purpose |
|----------|--------|---------|
| `src/app/icon.svg` | SVG | Next.js favicon (auto ico generation) |
| `src/app/apple-icon.png` | 180x180 PNG | Apple touch icon |
| `public/logos/icon-192.png` | 192x192 PNG | PWA manifest |
| `public/logos/icon-512.png` | 512x512 PNG | PWA manifest |
| Header component | SVG inline | Icon before "StudentPerks" in nav |
| OG image | Update background | Bolt as subtle watermark |

**Bolt style:** Minimal, 2-3 angles, no extra details. Rounded line caps. Slight tilt for energy.

**Header:** 20x20 bolt left of "StudentPerks" text, gradient fill.
